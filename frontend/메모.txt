1. 개발환경
시스템 서버 : Spring Boot 3.4.2
화면(Web) : React JS 19.v
WAS : 내장 톰캣 활용
DB : MySql 

2. 개요
사용자의 일정관리, TO-DO 관리 Web Application

3. 화면별 기능
1) 홈화면
 - 더미 데이터 적용 - 기존 데이터를 적용하시겠습니까? 문구 수정
 - TO-DO 기능 제공
    - 일정 다운로드 기능 제공(txt)

2) 할일(TO-DO) CRUD
 - 날짜 컬럼 관련 기본값 설정(등록일시, 수정일시)
 - 추가할 경우 캘린더에 동기화
 

3) 캘린더
 - 더미 데이터 적용시 날짜별 일정 동기화 오류 수정
 - To-Do - useLocation으로 URL 체크하여 일정 팝업에서 삭제 버튼 안보이게, 수정은 클릭시 Result 페이지로 이동

4) 검색
 - Result 페이지에서 검색시 결과 바뀌는 오류 수정
